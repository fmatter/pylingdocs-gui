# pylingdocs-gui

A rudimentary GUI editor for pylingdocs.

![License](https://img.shields.io/github/license/fmatter/pylingdocs-gui)

* Installation: `pip install pylingdocs-gui`
* Usage: run `pylingdocs-gui` in your pylingdocs project
